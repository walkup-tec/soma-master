# Projeto Soma

CRM baseado no código do **CRM Sinal Verde**, agora no repositório independente [walkup-tec/soma-master](https://github.com/walkup-tec/soma-master).

## Pasta local

`D:\Soma`

## Setup rápido

```bash
bun install
cp .dev.vars.example .dev.vars
# preencher secrets locais (ou copiar de D:\CRM-SinalVerde\.dev.vars se for o mesmo ambiente)
bun run dev
```
