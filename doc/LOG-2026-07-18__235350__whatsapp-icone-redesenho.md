# LOG — Ícone WhatsApp topbar: redesenho limpo

## Problema
Contorno borrado: stroke em path de marca preenchida + `animate-pulse` no traço.

## Solução
- Path estilo Lucide: MessageCircle + fone geométrico, `strokeWidth={2}`
- Pulso só no anel externo (não no ícone)
- Tamanho 18px

## Keywords
whatsapp, icon, topbar, lucide, outline
